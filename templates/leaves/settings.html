About Leave Types

Leave types define the different categories of leave available to employees. Each leave type has:

  • Default Days: The number of days allocated to each employee per year
  • Paid/Unpaid: Whether the leave is paid or unpaid
  • Approval Required: Whether the leave requires manager approval
  • Status: Active leave types are available for employees to request
